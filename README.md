*Untuk struktur,* halaman dibagi dua kolom dengan CSS Grid: panel kiri berisi form input (foto, nama, NRP, gugus, pilihan warna), panel kanan berisi preview kartu yang akan diunduh. Kartu ID sendiri dibuat sebagai elemen terpisah (#id-card) supaya bisa langsung ditangkap oleh html2canvas saat tombol download ditekan.  
*Untuk theming,* digunakan CSS custom properties (--accent-1, --accent-2, --pattern) yang nilainya berubah lewat class theme-pink atau theme-blue pada elemen kartu. Jadi tombol toggle warna cukup menukar class, dan semua gradient header/footer serta border foto otomatis ikut berubah tanpa perlu duplikasi style.  
*Untuk interaktivitas,* pendekatannya adalah live binding: setiap input teks (nama, nrp, gugus) punya event listener input yang langsung menulis nilainya ke elemen preview yang sesuai, sekaligus toggle class empty supaya placeholder "—" muncul kalau field kosong. Untuk foto, dipakai FileReader.readAsDataURL agar gambar yang dipilih user langsung dirender sebagai base64 di <img>, lalu ikon default disembunyikan.  
*Untuk download,* dipakai library html2canvas: elemen kartu dirender jadi canvas dengan scale 3x (biar hasilnya tajam/HD), lalu di-convert ke PNG via toDataURL dan diunduh otomatis dengan nama file yang sudah disanitasi dari input nama.
